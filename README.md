
## How It Works

Claude Code automatically reads `CLAUDE.md` from your project root on every session. No configuration needed.

**Example prompt:**
```
Build a SaaS landing page with hero, pricing, and testimonials
```

**Claude will:**
1. Choose a bold aesthetic direction
2. Select matching components from 21st.dev
3. Install them via CLI
4. Wire everything together with Next.js 15
5. Add scroll animations with Framer Motion
6. Ensure WCAG AA compliance
